Evaluation of web graphics

Wild Rockies Field Institute

Site Evaluated: WRFI

Purpose of the graphics on the page selected:

Very nice subtle photos to admire. Very plain and simple font and background, eye catching headings. Easy to access the places you are looking for. Nice hover setting for the navigation bar. The acedemics and cources section is very inviting, interesting pictures to look at and fun activities to explore such as hiking and canoeing

Guidelines for effective use

  1. Yes- Graphics fits witht the purpose , organization, andy style of page
  2. Yes- Avoids usng graphics with large file sizes that add to the load time of the page
  3. Yes- Graphics help guide the viewers focus to the important content on the page
  4. Yes- Avoids repetitive use of overly bright or potentinally ovnoxious images
  5. Yes- Avoids the use of graphics to convey textual content information
  6. Yes- Provides textual equivalent alternatives for graphic content
  7. Yes- When text is used with graphics, there is sufficient contrast between the text and background color

Graphic File Types

Pixles a minute area of illumination on a display screen, one of many from which an image is composed
Resolution number of pixels (individual points of color) contained on a display monitor
Lossy Compression Lose Data when it is compressed
Lossless Compression No data lost when it is compressed
JPEG Can display about 16.8 million colors, Lossy Compression Cannot be animated, Great for still images and photographs
PNG PNG-8:Can display 256 colors, PNG-24:Can display millions of colors, Lossless Compression, Better transparency than GIF, Cannot be animated, Good for icons;Can be used when an SVG format is not available
SVG Very scalable and great for responsive design, Can be modified using CSS and JavaScript, Smaller file size; depends on how complex the image is, Can be animated, Great for logos, icons, maps, flags, charts etc.,Not so good for photographs
GIF Can display 256 colors, Lossless Compression, Supports transparency, Great for simple animations, Good for graphics with flat colors and graphics without gradients
